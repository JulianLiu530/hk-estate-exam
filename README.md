# 🏠 香港地產代理牌照備考平台

> 為備考香港地產代理（E牌）的朋友而建，免費使用。

---

## 初心

備考 E 牌的時候，找不到一個好用的線上刷題平台——要麼題目太少，要麼沒有模擬考試功能，要麼介面難用到放棄。

考完之後發現，身邊很多人都有同樣的困擾，對備考資料的需求很大。

於是自己動手，用 Vibe Coding 的方式把這個平台做了出來，希望能幫到同樣在備考路上的朋友。

---

## 功能

### 📚 練習模式
按題目逐題作答，提交後即時顯示正確答案與詳細解析，適合鞏固知識點。

### 📝 模擬考試
還原真實 E 牌考試環境：
- **Part 1**（必答題）與 **Part 2**（個案分析）分開計時
- 通過條件：Part 1 ≥ 18/30，Part 2 ≥ 12/20
- 考試結束後顯示成績及逐題解析

### ❌ 錯題本
自動記錄做錯的題目，方便針對薄弱環節反覆練習。

### 📊 學習統計
可視化答題進度與正確率，清楚掌握自己的備考狀態。

### ☁️ 雲端同步
登入後進度自動同步，手機與電腦之間無縫切換。
